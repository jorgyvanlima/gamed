GAME D SOBRE A MÁQUINA

O GAME D é uma plataforma digital dedicada a resgatar e preservar a jogabilidade pura dos clássicos arcade. Sem instalações complexas, sem cadastros demorados. É só abrir, inserir a ficha e jogar.

Utilizando tecnologias modernas como HTML5 Canvas e Web Audio, nós recriamos a física, a estética visual das telas CRT e os efeitos sonoros característicos dos anos 80 e 90.

Sem Download

Jogue instantaneamente no computador ou celular diretamente do seu browser.

Suporte a Teclado & Toque

Controles responsivos configurados para teclado e botões virtuais na tela.

Áudio Sintetizado

Som retro gerado dinamicamente no seu navegador usando Web Audio API.

Você é Desenvolvedor de Jogos Retrô?

A GAME D está sempre procurando novos talentos para expandir nossa coleção. Se você tem um jogo feito em Canvas, WebGL ou emuladores JS, envie para nós!

PLATAFORMA RETRÔ DE JOGOS ONLINE

🎓 Contexto Acadêmico

Este projeto consiste em um trabalho prático desenvolvido pelo Grupo D como parte dos requisitos de avaliação da disciplina de desenvolvimento seguro na Pós-Graduação em Cibersegurança da Universidade Federal do Pará (UFPA).

O objetivo principal é demonstrar a construção de uma plataforma web interativa (Single Page Application) inspirada em serviços de streaming como o Avantiflix, mas focada em jogos retrô funcionais, aplicando boas práticas de segurança defensiva no front-end.

🕹️ Sobre a Cabine

O GAME D é construído sob uma forte estética de fliperama dos anos 80 e 90 (Synthwave/Cyberpunk). Ele emula o comportamento visual das telas clássicas com filtros CSS que geram linhas de varredura (scanlines), simulam o vidro abaulado do monitor e sintonizam chiptunes retrô em tempo real gerados via osciladores nativos do áudio web.

CONTROLES DE SEGURANÇA

Mitigação de Cross-Site Scripting (XSS)

Nenhuma entrada de dados externa é interpretada diretamente como código. Apelidos no ranking e palavras-chave na busca de jogos são sanitizados utilizando a propriedade nativa textContent do DOM. Tags de script inseridas são tratadas como texto comum, impedindo vulnerabilidades do tipo HTML Injection ou Stored XSS.

Encapsulamento de Escopo e Sandbox

Os motores dos jogos (Space Invaders, Blocks, Snake) são inicializados de forma independente utilizando orientação a objetos clássica. Os estados de pontuação e loops de frames estão contidos dentro do escopo léxico das classes de cada jogo, impedindo adulteração direta das variáveis de pontuação a partir do console global do browser.

Políticas de HTTPS e Cabeçalhos de Resposta (CSP/HSTS)

Configurado Nginx sob certificado TLS forçando conexões HTTPS na porta 443 e redirecionando a porta 80. O servidor entrega cabeçalhos anti-clickjacking (X-Frame-Options: DENY), anti-MIME-sniffing (X-Content-Type-Options: nosniff) e uma rigorosa Content-Security-Policy limitando conexões e fontes de mídias.

🛠️ Especificações da Máquina

HTML5 Semântico & Canvas

Renderização de sprites e malhas a 60 FPS através de animações baseadas no ciclo de varredura da GPU.

CSS3 Synthwave

Design responsivo, sombras de neon dinâmicas e camadas emuladoras de monitor CRT de fósforo.

Web Audio API Synth

Sintetizador de áudio retro integrado. Cria ondas sonoras quadradas e dente-de-serra sintéticas em tempo real.

Sem Dependências externas

Segurança reforçada ao evitar frameworks complexos. Código 100% sob auditoria própria.
